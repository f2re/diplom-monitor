# Plan.md - Проект "Weeks Until Diploma"

## 1. ТЕХНИЧЕСКОЕ ЗАДАНИЕ (Обновлено)

### 1.1 Описание
Веб-приложение для визуализации времени до защиты диплома. Сетка недель помогает осознать дедлайн и мотивирует к регулярной работе.

### 1.2 Целевой Стек
- **Backend**: FastAPI (Python 3.11+) + PostgreSQL.
- **Frontend**: Vue.js 3 (Vite, Composition API) + Tailwind CSS.
- **Инфраструктура**: Docker, GitHub Actions.

---

## 2. АРХИТЕКТУРА И СТАНДАРТЫ

### 2.1 Принципы разработки
- **KISS & DRY**: Минимализм в коде, переиспользование логики через Composables (Vue) и Services (FastAPI).
- **Mobile-First**: Сетка и интерфейс оптимизированы для экранов от 320px.
- **Type Safety**: Использование Pydantic (Backend) и TypeScript (Frontend - опционально, по согласованию).

### 2.2 Структура данных
- **Users**: Профиль, настройки дат (start, deadline), выбранный эмодзи.
- **WeekProgress**: Статус недели (выполнено/нет), текстовая заметка.
- **SpecialPeriods**: Отпуска, командировки и другие исключения.
